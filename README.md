# Sliding-Puzzle-Game
3×3 Sliding Puzzle Game

Developed to improve logical thinking and problem-solving skills. In this game, numbers tiles are arranged in a 3×3 grid with one empty space. The player moves the tiles into the empty space to arrange them in the correct order. The project provides an interactive and user-friendly interface with smooth tile movement and game controls. It mainly focuses on game logic, puzzle-solving techniques, and user interaction. The game is simple, engaging, and helpful for improving concentration and analytical skills.
